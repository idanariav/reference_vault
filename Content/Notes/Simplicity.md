# Simplicity

## Note

Simplicity is the idea that (Related:: [[Addition by subtraction|less is more]]). That by (Related:: [[Distillment|tripping]]) something down to the core we can get a much more refined, understandable and implementable idea. as they say "kiss" -> keep it simple, stupid. 
The added value of simplicity is that the main constraint is a low level of (Related:: [[Friction]]). If something is difficult to use or understand, than it's not simple enough. This is also true in it's reverse, like the (Jump:: [[Feynman Technique]]) that says that if you can't explain it to others (in a simple way) that you don't understand it well enough.
